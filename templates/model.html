Sentiment Analysis

Vader Sentiment 

  • VADER (Valence Aware Dictionary and sEntiment Reasoner) is a rule-based sentiment analysis tool that is specifically attuned to sentiments expressed in social media. It is based on a lexicon of sentiment-related words and phrases that have been rated by human annotators as either positive or negative.

Roberta Sentiment

  • Roberta is a pre-trained deep neural network model for natural language processing (NLP). Roberta achieves state-of-the-art performance on many standard benchmarks for sentiment analysis and it is trained on a large, diverse corpus of text data using a multi-task learning framework that involves training on multiple NLP tasks simultaneously.

Classification

  • A classification model is a type of machine learning model that is used to classify or categorize input data into one of several predefined categories or classes. The model is trained on a labeled dataset, meaning that each data point in the training set is already categorized with the correct class label. Once the model has been trained, it can be used to classify new, unseen data into one of the classes it has learned.

Emotion Detection

  • Emotion detection from text is a natural language processing task that involves identifying the emotions expressed in a given piece of text, such as a tweet, email, or customer review. The goal is to classify the text into one or more emotion categories, such as happy, sad, angry, or neutral, based on the words used and their context.
